    

	Программа предназначена для парсинга выписки по звонкам компании МТС(можно заказать в личном кабинете) в формате html.

Перед началом работы необходимо установить базы данных Mysql и в файле config.py прописать начальные данные для подключения
и настройки работы программы(например время работы сотрудника, для проверки звонков в рабочее время).
Также в файле "Номера телефонов - фамилии агентов.txt" необходимо прописать: номер_телефона имя_сотрудника. Как по шаблону
Шаблон написания номеров - 79131234567
	
Необходимо чтобы файл выписки из личного кабинета, например "ЛС254392207369_номер79130112501_с_01042019_по_26042019.html" такой,
лежал в корне папки с программой.

Если номеров будет несколько, необходимо, в config.py изменить big_base = 1. Тогда программа выдаст список из количества разговоров 
и потраченного интернета за период всех сотрудников в порядке убывания. Если big_base = 0, то в корне должен лежать 1 файл html и 
для него будет расширеная информация по каждому номеру на который звонил абонент, а также расширеная информация по трафику.

Все результаты храняться в файле 'Результаты.txt'
